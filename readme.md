# Retornando Propriedades do Sistema Operacional com Node.js
## Vantagens do Node
- Flexibilidade
- Leveza
- Escalabilidade
- Redução de Custos
- Maior repositório do Mundo - NPM
- Mesma linguagem no frontend e no backend

## Conceitos básicos do node: 
- Variáveis - let, var e const;
- Objetos   - elementos que agrupam atributos e conteúdos;
- Funções   - blocos de código com instruções a serem exexutadas;
  
## Blocking, Non Blocking e Event loop
- Blocking: síncrono pois depende que todas as requisições estejam prontas;
- Non Blocking: assíncrono, a entrega é feita mesmo sem todas as requisições estarem prontas;
- Event-loop: execuçãod e requisições de forma assíncorna.

## NPM - node Package Manager
 Como o nome sugere é a ferramenta de gerenciamenro de Pacotes do Node para o seu projeto.
